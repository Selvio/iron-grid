Iron Grid — battlefield art provenance
======================================

STATUS: PRIVATE PROTOTYPE ONLY. DO NOT PUBLISH OR DEPLOY PUBLICLY.

The sprites in this directory are ripped from the Game Boy Advance
"Advance Wars" games. They are the intellectual property of Nintendo
and Intelligent Systems, and are used here solely as placeholder art
while the game's own art is produced.

  terrain/    autotile sets (16 px tiles)
  units/      per-faction unit sheets (sprites / air / sea)
  buildings/  colored_buildings.png — property art in 5 ownership colors
  ui/         things.png — HUD stat icons
  fx/         death / missile / attack effects

Rips credited by the source sheets:
  - Dr. Phileas Fragg  (AW1Sprites)
  - Grim, http://sdb.neuropod.net  (air and naval unit sheets)
  - Rogultgot  (ground unit sheets)

Imported from the GPL-licensed student project
https://github.com/Peaupote/advance-war (assets/). The Java code there
is GPLv3; the artwork is not that project's to license, and neither is
it ours.

Consequences, binding on this repository:

  1. No public deployment, distribution, or asset download may ship
     these files. Anything user-facing must first replace them with
     original or properly licensed art.
  2. The render pipeline reads geometry from a generated atlas manifest
     (app/lib/render/atlas.generated.ts), so swapping the pack is a
     matter of regenerating that manifest — see
     docs/decisions/0005-advance-wars-asset-pack.md.

The previous pack (Pangea Wars by Aleksandr Makarov / @IKnowKingRabbit,
commercially licensed) was removed in the same change and remains in
git history if it needs to be restored.
